整理出 Notion 快捷鍵并分类

先前試用Notion ai會出一部份，另一部份是練習的操作

以下是 Notion 中常用的一些快捷键，并按照分类进行整理：

常规操作

  • Ctrl/Cmd + N: 新建页面
  • Ctrl/Cmd + S: 保存当前页面
  • Ctrl/Cmd + Z: 撤销上一步操作
  • Ctrl/Cmd + Shift + Z: 重做上一步操作
  • Ctrl/Cmd + X: 剪切选中的内容
  • Ctrl/Cmd + C: 复制选中的内容
  • Ctrl/Cmd + V: 粘贴选中的内容
  • Ctrl/Cmd + A: 全选当前页面内容
  • Ctrl/Cmd + F: 查找页面中的关键字
  • Ctrl/Cmd + SHIFT + N: 開新頁面(Notion 電腦版)
  • Ctrl/Cmd + SHIFT + L: 背景顏色切換
  • Ctrl/Cmd + +: 畫面放大
  • Ctrl/Cmd + -: 畫面縮小
  • Ctrl/Cmd + A: 全選文字
  • Ctrl/Cmd + D: 再製(Duplicate)目前區塊
  • Alt + J: 開啟資料庫項目時，切換到下一個 ****
  • Alt + K: 開啟資料庫項目時，切換到上一個 ****
  • Cmd/Alt + Ctrl + Shift + F: 將頁面加入我的最愛

页面内编辑

  • /: 快速添加各种内容块
  • Ctrl/Cmd + Enter: 添加新行
  • Ctrl/Cmd + B: 加粗选中的内容
  • Ctrl/Cmd + I: 斜体选中的内容
  • Ctrl/Cmd + D: 删除选中的内容
  • Ctrl/Cmd + K: 添加链接
  • Ctrl/Cmd + Shift + M: 添加评论
  • Ctrl/Cmd + Shift + L: 添加待办事项
  • Ctrl/Cmd + Shift + T: 添加表格
  • MAC: Ctrl + command + Space : 開啟emoji
  • WIN: win + >. : 開啟emoji
  • Ctrl/Cmd + ↑↓←→: 延伸選取範圍
  • Ctrl/Cmd + Shift +↑↓: 上下移動區塊
  • Tab: 縮排(階層)
  • Shift + Tab: 取消縮排(階層)
  • Ctrl/Cmd +?: 開啟該區塊設定清單
  • Option/Alt + ⎕: 拖曳區塊時可複製區塊
  • [ + [: 選擇連結另一個頁面(新增/原有)
  • @: 標註日期、人員、頁面
  • 選圖片後+ Space: 將圖片放大到全螢幕

页面间切换

  • Ctrl/Cmd + 1: 切换到左侧导航栏
  • Ctrl/Cmd + 2: 切换到页面列表
  • Ctrl/Cmd + 3: 切换到收藏夹
  • Ctrl/Cmd + 4: 切换到最近使用的页面
  • Ctrl/Cmd + 5: 切换到垃圾桶
  • Ctrl/Cmd + {: 回到上一頁
  • Ctrl/Cmd + }: 回到下一頁
  • Ctrl/Cmd + Shift + U: 頁面結構上升到上一層
  • Ctrl/Cmd + Option/Alt + T: 展開/收合折疊列表

搜索

  • Ctrl/Cmd + P: 快速搜索页面

以上是 Notion 中常用的一些快捷键，希望可以帮助到您更高效地使用 Notion。

文字樣式

  • 用符號包圍文字

SHIFT + 88: 文字加粗

SHIFT + 8 : 文字斜體

:code`

~~ : 刪除線

  • 選取後快捷鍵

Ctrl/Cmd + Shift + M: 加註解

Ctrl/Cmd + U: 文字底線

Ctrl/Cmd + U: 文字粗體

Ctrl/Cmd + I: 文字斜體

Ctrl/Cmd + Shift + S: 文字刪除線

Ctrl/Cmd + K: 文字加連結 **

Ctrl/Cmd + E: Ｍ

Ctrl/Cmd + Shift + H: 套用最後一次文字設定顏色

區塊樣式

  • SHIFT + * + SPACE
  • []中括號
  1. 數字 ＋ . + SPACE

# 一個

＃＃兩個

＃＃＃三個

    • SPACE 摺疊列表

“ + SPACE 引言

  • *3 分隔線

Ctrl/Cmd + Option/Shift + 1

Ctrl/Cmd + Option/Shift + 2

Ctrl/Cmd + Option/Shift + 3

  • Ctrl/Cmd + Option/Shift + 4
  • Ctrl/Cmd + Option/Shift + 5
  1. Ctrl/Cmd + Option/Shift + 6
  • Ctrl/Cmd + Option/Shift + 7

Ctrl/Cmd + Option/Shift + 9

Ctrl/Cmd + Option/Shift + 0
